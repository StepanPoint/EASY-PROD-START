# 🎙️ EASY PROD START

Генератор текстов для треков на нейросети, которая работает прямо в браузере.

## Возможности

- 🤖 Нейросеть работает ЛОКАЛЬНО — не нужен интернет
- 🎛️ Настройка BPM, стиля, темы текста
- 🔺 Подстазки (выделение ударений)
- 📋 Копирование результата

## Как запустить

1. Открыть index.html в браузере Chrome/Edge
2. Подождать загрузки модели (первые 5-10 минут)
3. Выбрать настройки и нажать "Сгенерировать текст"
